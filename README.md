# Memory Classifier - فارز IC

موقع لتصنيف رقائق الذاكرة (EMMC, UFS, NAND Flash) للموبايل.

## المميزات
- تصوير الرقاقة بالكاميرا والتعرف التلقائي
- إدخال الكود يدوياً
- قاعدة بيانات شاملة للرقائق
- تصنيف تلقائي (عادي / زجاجي)
- عرض السعة والرام

## التشغيل

```bash
npm install
npm start
```

## Environment Variables

```
ANTHROPIC_API_KEY=your_api_key_here
PORT=3000
```
